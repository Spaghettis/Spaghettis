
# Spaghettis

    Yet another (AT YOUR OWN RISK) fork of Pure Data < http://msp.ucsd.edu/ >.

Platforms:

    Tested on Ubuntu 18.04, macOS 10.14, and Raspbian (RPI4).
    
Dependencies:

    On GNU/Linux following packages are required:
    
    - tk8.6
    - tcl8.6
    - libasound2-dev
    - libjack-jackd2-dev

Build:

    Open a terminal and run the "Spaghettis/build.sh" script.
    Launch the "Application/Spaghettis.app" on macOS.
    On GNU/Linux execute "Spaghettis/build/bin/spaghettis".

Residual:

    ~/Library/Application Support/Spaghettis/
    ~/.config/spaghettis/
